# **Personal Expense Control System with JSON/CSV Persistence**

The **most complete, professional, and educational** system for managing your personal finances — all created by a student!

### **What this program does (all in a single file)**

* Register **income** (salary, freelance work, etc.) and **expenses** (food, Netflix, transportation…)
* Automatic validation: it doesn’t allow negative amounts or invalid types
* Calculate your **current balance** in real time
* View a **category summary** showing how much you spent on each item
* Save and load all your data in **JSON** and **CSV** (real persistence!)
* Compatible with Excel, Google Sheets, and any spreadsheet program
* 100% professional code with **advanced OOP**, typing, exceptions, and clean structure
